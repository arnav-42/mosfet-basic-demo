# MOSFET Visualizer

Interactive MOSFET drain current characteristics visualizer with real-time parameter adjustment and draggable interface cards.

## Stack

- **TypeScript**
- **Vite**
- **Plotly.js** - plotting library
- **Vanilla HTML/CSS** - No frameworks


## Quick Start

```bash
npm install
npm run dev
```

Open in browser.

## Build

```bash
npm run build
```

Outputs to `dist/` directory.
